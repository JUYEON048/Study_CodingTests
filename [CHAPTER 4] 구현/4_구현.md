- **구현(Implementation)**
  - 머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정.
- 흔히 풀이를 떠올리는 것은 쉽지만 소스코드로 옮기기 어려운 문제를 지칭.
  
    
 - 일반적으로 알고리즘 문제에서의 2차원 공간은 **행렬**(Matrix)의 의미로 사용.

- 시뮬레이션 및 완전 탐색 문제에서는 2차원 공간에서의 **방향 벡터**가 자주 활용.

    ''' python3
    # 동, 북, 서, 남
    dx = [0, -1, 0, 1]  # 행의 변화량
    dy = [1, 0, -1, 0]  # 열의 변화량
    
    #현재 위치
    x, y = 2, 2
    
    for i in range(4):
        # 다음 위치
        nx = x + dx[i]
        ny = y+ dy[i]
        print(nx, ny)
    '''
